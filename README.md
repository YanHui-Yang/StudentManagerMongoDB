### 基于Servlet写的学生管理系统，前端是Layuimini，数据库采用MongoDB，实现增删改，模糊查询，分页查询的功能



### 运行方式

在mongo执行

```sql
use student_manager

db.user.insert({"username": "admin", "password": "123456"})
```
